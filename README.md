# 🚀 Aniket's Portfolio

A modern, dynamic personal portfolio website built with React and Vite. Showcase your skills, projects, and coding profiles with an interactive matrix background and smooth animations.

## ✨ Features

- **Hero Section** - Eye-catching introduction with animated effects
- **About Section** - Professional background and personality highlights
- **Skills Display** - Organized showcase of technical skills
- **Projects Portfolio** - Detailed project cards with descriptions and links
- **Coding Profiles** - Links to GitHub, LeetCode, and other coding platforms
- **Matrix Background** - Animated matrix-style background for visual appeal
- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Fast Performance** - Built with Vite for lightning-fast development and production builds

## 🛠️ Tech Stack

- **Frontend Framework**: React 18+
- **Build Tool**: Vite
- **Styling**: CSS
- **Code Quality**: ESLint
- **Package Manager**: npm

## 📦 Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd aniket-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## 📁 Project Structure

```
src/
├── components/
│   ├── Hero.jsx           # Landing section
│   ├── About.jsx          # About me section
│   ├── Skills.jsx         # Skills showcase
│   ├── Projects.jsx       # Projects portfolio
│   ├── CodingProfiles.jsx # External profile links
│   ├── Footer.jsx         # Footer with contact info

└── index.css              # Global styles
```

## 🚀 Getting Started

1. Customize the content in each component
2. Add your projects to the `Projects.jsx` component
3. Update your skills in `Skills.jsx`
4. Add your coding profile links in `CodingProfiles.jsx`
5. Personalize the Hero and About sections
